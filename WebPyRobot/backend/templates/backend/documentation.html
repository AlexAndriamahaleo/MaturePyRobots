Documentation

  • getSize()

    Permet de connaître la taille de la map.

    Retourne une valeur numérique.
  • getTankId()

    Permet de connaître l'identifiant de votre tank.

    Retourne une variable de type TankID.
  • getEnemyTankId()

    Permet de connaître l'identifiant du tank ennemie.

    Retourne une variable de type TankID.
  • getCellDistance(NumCell1, NumCell2)

    Permet de connaître la distance entre la cellule NumCell1 et la cellule NumCell2.

    Retourne une valeur numérique. Hint: imaginez la distance entre deux points dans un repère orthonormé.
  • getCellFromXY(X, Y)

    Permet de connaître le numéro d'une cellule à partir d'une coordonnée X et Y.

    Retourne une variable de type NumCell.
  • getCellPosX(NumCell)

    Permet de connaître l'ordonnée X d'une cellule NumCell.

    Retourne l'abscisse d'une cellule NumCell.
  • getCellPosY(NumCell)

    Permet de connaître l'ordonnée Y d'une cellule NumCell.

    Retourne l'ordonnée d'une cellule NumCell.
  • getPosition(TankID)

    Permet de connaître le numéro de la cellule sur laquelle se trouve le tank qui a pour identifiant TankID.

    Retourne une variable de type NumCell.
  • getLife(TankID)

    Permet de connaître la vie restante du tank qui a pour identifiant TankID

    Retourne une valeur numérique.
  • getPM(TankID)

    Permet de connaître le nombre de Point de Mouvement actuel (ou restant) du tank d'identifiant TankID.

    Retourne une valeur numérique.
  • getPA(TankID)

    Permet de connaître le nombre de Point d'Action actuel (ou restant) du tank d'identifiant TankID.

    Retounr une valeur numérique.
  • getRange(TankId)

    Permet de connaître la portée du tank d'identifiant TankID.

    Retourne une valeur numérique.
  • moveTank(NumCell)

    Fonction permettant de déplacer son tank vers une cellule NumCell. Si celle-ci n'est pas à sa portée (par rapport aux nombres de PM), le tank se déplace vers une cellule proche de celle désié.

    Action réitérable en fonction des paramètres courant.
  • shoot()

    Fonction permettant de tirer sur le tank ennemi.

    Action réitérable en fonction des paramètres courant.
